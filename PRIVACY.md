# Privacy Policy (FreshBook)

This add-on does **not** collect, store, or transmit any personal data.

- The extension only runs on `facebook.com` pages.
- It modifies the page DOM locally in your browser to hide selected elements.
- Settings are stored using Firefox `storage.sync` (or `storage.local` depending on your Firefox configuration).

No analytics, no trackers, no external network calls.
Fully open source, so you can control the sources
